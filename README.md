# FOOD EXPLORER
Aplicação de um cardápio digital para um restaurante fictício, conhecido como FoodExplorer.<br>
O projeto será divido em duas partes: `Front-end` e `Back-end`

## Instalação

```bash
# Clone o projeto

# Faça a instalação das depêndencias
  npm install

# Execute o projeto
  npm run dev
```
